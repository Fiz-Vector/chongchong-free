## ❓介绍

虫虫钢琴为你提供优质的免费钢琴谱资源（❌）

虫虫钢琴为你提供低技的术付费钢琴资源（✔）

`因为🐛🐛钢琴没有登陆验证，所以这个工具让你能以白嫖🐛🐛钢琴的MIDI力！`
`（前提是能下载（有AI乐谱））`

## 🚀功能

* 虫虫钢琴下载器是一个免费下载虫钢琴谱的软件，能获取琴谱的链接并解析琴谱的内容，并导出文件。
* 导出PDF琴谱
* 导出MIDI文件
* 导出MP3文件

## 🚗运行

1) 运行 `npm install`
2) 运行 `node chongchong-free.js`

## ⚙参数

1) `-i <琴谱ID>` （必要）
2) `-o <输出目录>` 默认为output（可选）
3) `-p` 输出原始PDF 默认不输出（可选）
4) `-m` 输出MP3 默认不输出（可选）
5) `-d` 详细输出 默认为关（可选）
